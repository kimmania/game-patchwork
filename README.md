# Patchwork

A PWA puzzle game where you manipulate real-world data structures — network topologies, database schemas, dependency graphs, and more — to solve routing, integrity, and optimization challenges.

Built with Vite + TypeScript + `vite-plugin-pwa`, deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main`; GitHub Actions builds and deploys to Pages.
